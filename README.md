# Хуульчийн Тест (Lawyer Test) - Вэб Аппликейшн (Flet Web)

Энэхүү вэб аппликейшнийг Python хэлний Flet фреймворк болон WebAssembly (Pyodide) технологи ашиглан хөгжүүлсэн бөгөөд хэрэглэгч ямар нэгэн суулгалт хийхгүйгээр шууд хөтчөөрөө шалгалт өгөх, үр дүнгээ кирилл үсгийн зөв форматтай PDF тайлан болгон татах боломжтой.

## Үндсэн онцлогууд:
* **Бүрэн статик вэб (Serverless Wasm)**: Ямар нэгэн backend сервер шаардахгүйгээр хэрэглэгчийн хөтөч дээр шууд ажиллана. Хурдан ажиллагаатай бөгөөд үнэгүй статик серверүүд дээр байршуулах боломжтой.
* **Зөөврийн PDF таталт**: Шалгалтын үр дүнг хөтчийн Base64 таталтын урсгал ашиглан утас болон компьютер руугаа алдаагүй шууд татаж авна.
* **Сайжруулсан зохицол (Safe Area & Word Wrap)**: Гар утасны дээд талын батерей болон системийн мэдээллүүдтэй давхцахгүй бөгөөд урт хариултууд дэлгэцийн өргөнд зохицон автоматаар доош шилжинэ.

---

## Хэрхэн локалоор ажиллуулах вэ?

Хэрэв та өөрийн компьютер дээр локалоор ажиллуулж туршихыг хүсвэл дараах алхмуудыг хийнэ үү:

1. **Шаардлагатай сангуудыг суулгах**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Статик вэб сервер ажиллуулах**:
   Энэхүү хавтас нь бэлэн статик файлуудаас бүрдэх тул Python-ий суулгасан серверээр шууд ажиллуулж болно:
   ```bash
   python -m http.server 8000
   ```
   Уг тушаалыг ажиллуулсны дараа хөтчөөрөө `http://localhost:8000` хаяг руу орж ашиглана уу.
